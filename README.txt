BARR AUTOMATION CO. — WEBSITE
=============================

FILES
- index.html ............ Main Barr Automation Co. agency site (open this)
- barr-digital.html ..... Earlier lead-gen site version (optional / reference)

HOW TO PREVIEW
- Double-click index.html to open it in any web browser.

BEFORE GOING LIVE (edit index.html)
1. Contact form: connect to your email / Formspree / GoHighLevel
   to actually receive leads (currently a demo form).
2. Portfolio: swap the sample project mockups for real client work.
3. Testimonials: replace placeholder reviews with real, approved quotes.
4. Contact info: update email, phone, and service area in the
   Contact section and footer.

HOSTING
- Upload index.html to any host (Netlify, Vercel, Squarespace
  code block, GoDaddy, etc.). It's a single self-contained file
  with no build step required.

Fonts load from Google Fonts, so an internet connection is needed
for the typography to display exactly as designed.
